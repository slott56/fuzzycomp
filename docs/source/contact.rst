Contact
=======

Bugs and Feature requests
-------------------------
To report bugs or to request the addition of a new feature, please use the dedicated issue tracker for the project found at https://code.google.com/p/fuzzycomp/issues/list.

Try to be as specific as possible when describing your bug/feature, preferably you should provide a test case with input data, expected output data and actual result received.


General
-------

To get in contact with the me regarding the project with questions, suggested improvements or any other reason, please contact me at fuzzycomp@googlegroups.com or follow me on twitter @fuzzycode.
